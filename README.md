Генератор случайных цитат

Информация об авторе
Имя:Каюмов Марсель

Описание
Генератор случайных цитат — это настольное приложение с графическим интерфейсом пользователя, созданное на Python и tkinter, которое генерирует случайные цитаты из предопределенной коллекции. Приложение позволяет пользователям:

- Генерировать случайные цитаты одним щелчком мыши
- Просматривать историю цитат с временными метками
- Фильтровать цитаты по автору и теме
- Добавлять новые цитаты в коллекцию
- Автоматически сохранять и загружать историю цитат

Функции
- Генерация случайных цитат: Нажмите кнопку, чтобы отобразить случайную цитату с указанием автора и темы
- Отслеживание истории: Все сгенерированные цитаты сохраняются с временными метками
- Фильтрация: Фильтруйте цитаты по автору или теме, чтобы получить цитаты из определенных источников или тем
- Управление цитатами: Добавляйте новые цитаты в коллекцию с проверкой
- Постоянное хранение: Цитаты и история сохраняются в JSON-файлы
- Удобный интерфейс: Чистый и интуитивно понятный графический интерфейс с корректной обработкой ошибок

Требования
- Python 3.6 или выше
- tkinter (обычно входит в состав Python)
- Дополнительные внешние библиотеки не требуются
- Установка
