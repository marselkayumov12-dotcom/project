## Описание проекта

**Random Password Generator** - это десктопное приложение для генерации безопасных паролей с графическим интерфейсом. Приложение позволяет настраивать параметры пароля, проверять его сложность и сохранять историю сгенерированных паролей в JSON-файл.

### Основные возможности

- 🔐 **Генерация паролей** с настраиваемыми параметрами:
  - Длина от 4 до 128 символов
  - Строчные и заглавные буквы
  - Цифры
  - Специальные символы (!@#$%^&* и др.)

- 📊 **Оценка сложности** пароля:
  - Очень сложный (5/5)
  - Сложный (4/5)
  - Средний (3/5)
  - Слабый (2/5 и менее)

- 💾 **История паролей**:
  - Автоматическое сохранение в JSON
  - Просмотр истории
  - Копирование паролей из истории
  - Очистка истории

- ⚡ **Быстрые настройки**:
  - Слабый (8 символов, только строчные)
  - Средний (12 символов, буквы+цифры)
  - Сильный (16 символов, все типы)
  - Максимальный (32 символа, все типы)

- 📋 **Дополнительные функции**:
  - Генерация 5 паролей одновременно
  - Копирование в буфер обмена
  - Сохранение паролей в историю

## Технологии

- Python 3.7+
- Tkinter (графический интерфейс)
- JSON (хранение истории)
- random (генерация паролей)
- unittest (модульное тестирование)
